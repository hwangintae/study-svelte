# 2. 스벨트 개발 환경 구축

```text
.
├── node_modules 
├── public
│   ├── favicon.png
│   ├── global.css
│   ├── index.html
├── script
├── src
│   ├── App.svelte
│   └── main.js
├── package.json
└── rollup.config.js
```

| 폴더/파일            | 설명                 |
| ---------------- | ------------------ |
| node_modules     | 설치된 패키지가 위치한 폴더    |
| public           | 빌드 결과              |
| script           | TS 설정 파일           |
| src              | 실제 프로젝트 코드가 들어갈 폴더 |
| package.json     | 설치된 패키지 정보         |
| rollup.config.js | rollup 설정 파일       |
